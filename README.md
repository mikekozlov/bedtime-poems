# Перед сном · тёплые стихи

Интерактивная антология из **47 стихов** о ночи, нежности, тишине и лёгком флирте — **8 стран** (Россия, Германия, Франция, Англия, Италия, Персия, Китай, Япония). Чтобы перечитать один перед сном и уснуть с тёплой улыбкой.

**Живая страница:** https://mikekozlov.github.io/bedtime-poems/

## Что внутри
- Вкладка‑оглавление + отдельная вкладка на каждую страну.
- Тёмная тема по умолчанию, режим чтения, поиск, фильтр по настроению, избранное (♥), «🎲 Случайный», регулировка шрифта.
- Один самодостаточный `index.html` — работает офлайн, без интернета и внешних зависимостей.

## Честно об источниках
- Все тексты — в общественном достоянии; каждый сверен с реальной страницей‑источником (ссылка стоит на карточке). Ничего не переведено «на ходу» и не выдумано.
- «Топ мировой поэзии» здесь — кураторский выбор под настроение «перед сном», а не объективный рейтинг.
- Иностранные стихи даны в public‑domain переводах (Фет, Бальмонт, Брюсов, Сологуб, Дельвиг, Мин, Кузмин, Нович и др.); страна указана по родине автора оригинала, переводчик — отдельно.
- Персидский раздел — фетовский цикл «Из Гафиза»: не буквальный Хафиз, а его знаменитые вольные подражания (о чём сказано в каждой карточке)
